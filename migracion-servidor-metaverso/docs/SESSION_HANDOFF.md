# Reanudacion de sesion

## Orden para retomar
1. Leer `docs/PLAN.md`.
2. Revisar `docs/CHECKPOINTS.md`.
3. Abrir `docs/PROGRESS.md`.
4. Continuar desde el ultimo punto no completado.

## Regla de continuidad
Antes de empezar una nueva tarea, registrar en `docs/PROGRESS.md`:
- que se hizo en la sesion anterior,
- en que punto se quedo,
- cual es la siguiente accion.

## Disciplina de trabajo
- No avanzar de checkpoint sin dejar evidencia escrita.
- No mezclar tareas nuevas con tareas ya cerradas sin registrar el cambio.
- Si aparece una decision de arquitectura, anotarla antes de seguir.
