# Registro de avances

## Formato de registro
Usar siempre una entrada por sesion o por hito relevante.

```text
Fecha: YYYY-MM-DD
Sesion: breve descripcion
Avance:
- que se hizo
- que quedo listo
Bloqueos:
- si los hay
Siguiente paso:
- proxima accion concreta
```
